# 🏏 IPL Data Analysis (EDA Project)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Indian Premier League (IPL) dataset to uncover insights about teams, players, and match trends.

The analysis is done using **Python, Pandas, and Matplotlib**, focusing on real-world data analysis and visualization skills.

---

## 🎯 Objectives

* Analyze team performance across seasons
* Calculate win percentage
* Identify top-performing players
* Understand venue-based trends
* Compare powerplay vs death overs performance

---

## 📂 Dataset

The dataset is not included in this repository due to size limitations.

You can download the dataset from Kaggle:
👉 https://www.kaggle.com/datasets

---

## 📊 Key Analysis

### 🏆 Team Analysis

* Matches played per team per season
* Win percentage calculation
* Team performance comparison

### 👤 Player Analysis

* Top batsmen by total runs
* Strike rate analysis (minimum 500 runs)
* Boundary analysis (4s and 6s)

### 🏟️ Venue Analysis

* Highest scoring venues
* Average first innings score
* Boundary frequency by stadium

### ⏱️ Match Insights

* Powerplay vs death overs performance
* Wicket distribution trends
